Extends Contact Form 7 Email Validation.

== Description ==

The plugin extends Contact Form 7 validations with SMTP verification.

= Features =
* SMTP commands logging

== Installation ==

= Installing CF7 SMTP Verify Plugin =

* Download the plugin zip file.
* Login to your WordPress Admin. Click on Plugins | Add New from the left hand menu.
* Click on the "Upload" option, then click "Choose File" to select the zip file from your computer. Once selected, press "OK" and press the "Install Now" button.
* Activate the plugin.

When activated, it will hook to any form that contains a mandatory email field and try to verify it.
